## 五、费用优化建议

### 5.1 一期优化（目标：$40-50/月）

**优化措施：**

1. **使用Lightsail免费套餐**
   - 第一个月免费试用
   - 节省：$20

2. **S3生命周期策略**
   - 30天后转IA（Infrequent Access）
   - 节省：约40%存储成本

3. **CloudFront缓存优化**
   - 静态资源1年缓存
   - 减少回源请求

4. **应用内Redis**
   - 一期不使用独立Redis
   - 节省：$5/月

5. **日志压缩**
   - 使用gzip压缩日志
   - 减少50%日志体积

**优化后一期费用：$40-45/月**

### 5.2 二期优化（目标：$55-65/月）

**优化措施：**

1. **Reserved Instances**
   - 购买1年期预留实例
   - 节省：约30%

2. **CDN优化**
   - 图片压缩（WebP格式）
   - 视频转码（H.264）

3. **数据库优化**
   - 定期清理旧数据
   - 索引优化

**优化后二期费用：$55-65/月**

### 5.3 成本监控

**设置CloudWatch告警：**

1. **月度费用告警**
   - 阈值：$60（一期）、$80（二期）
   - 通知：邮件

2. **资源使用告警**
   - CPU > 80% 持续5分钟
   - 内存 > 85% 持续5分钟
   - 磁盘 > 80%

3. **流量异常告警**
   - 出站流量异常增长

## 七、部署流程

### 7.1 初始部署

```bash
# 1. 创建Lightsail实例
# 选择：OS Only > Ubuntu 22.04 LTS
# 方案：$20/月

# 2. SSH连接
ssh ubuntu@your-instance-ip

# 3. 安装依赖
sudo apt update
sudo apt install -y nginx nodejs npm postgresql-client redis-tools

# 4. 安装Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

# 5. 安装PM2
sudo npm install -g pm2

# 6. 配置Nginx
sudo nano /etc/nginx/sites-available/schoolmanagement
# 配置反向代理到localhost:3000

# 7. 克隆代码
git clone https://github.com/yourorg/schoolmanagement.git
cd schoolmanagement

# 8. 安装依赖
npm install

# 9. 配置环境变量
cp .env.example .env
nano .env

# 10. 运行数据库迁移
npm run migration:run

# 11. 构建应用
npm run build

# 12. 启动应用
pm2 start npm --name "schoolmanagement" -- run start:prod
pm2 save
pm2 startup

# 13. 重启Nginx
sudo systemctl restart nginx
```

### 7.2 配置SSL

```bash
# 使用Let's Encrypt（免费）
sudo apt install -y certbot python3-certbot-nginx
sudo certbot --nginx -d schoolmanagement.com -d www.schoolmanagement.com
```

### 7.3 配置自动备份

```bash
# Lightsail数据库自动备份（控制台配置）
# 应用代码备份：GitHub + 定期导出
# S3文件：版本控制
```

### 7.4 CI/CD配置

```yaml
# .github/workflows/deploy.yml
name: Deploy to Production

on:
  push:
    branches: [main]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: 20
          
      - name: Install dependencies
        run: npm ci
        
      - name: Run tests
        run: npm test
        
      - name: Build
        run: npm run build
        
      - name: Deploy to Lightsail
        uses: appleboy/ssh-action@master
        with:
          host: ${{ secrets.HOST }}
          username: ${{ secrets.USERNAME }}
          key: ${{ secrets.SSH_KEY }}
          script: |
            cd /home/ubuntu/schoolmanagement
            git pull origin main
            npm ci --production
            npm run migration:run
            pm2 restart schoolmanagement
```

## 八、监控与维护

### 8.1 日常监控

**监控指标：**

1. **应用健康**
   - 响应时间 < 500ms
   - 错误率 < 1%
   - 可用性 > 99.5%

2. **服务器资源**
   - CPU < 70%
   - 内存 < 80%
   - 磁盘 < 80%

3. **数据库**
   - 连接数 < 80
   - 慢查询 < 10/分钟
   - 复制延迟 < 1秒

### 8.2 定期维护

**每周：**
- 检查错误日志
- 检查备份状态
- 检查安全更新

**每月：**
- 检查费用账单
- 清理旧日志
- 优化数据库

**每季度：**
- 安全审计
- 性能优化
- 容量规划

### 8.3 故障恢复

**故障场景：**

1. **应用崩溃**
   - PM2自动重启
   - 检查日志定位问题

2. **数据库故障**
   - Lightsail自动故障转移
   - 从备份恢复

3. **服务器故障**
   - 创建新实例
   - 从Git重新部署
