# 数据库设计文档

**版本：** 1.0  
**日期：** 2026-03-17  
**数据库：** PostgreSQL 16.x

---

## 一、数据库概述

### 1.1 设计原则

- **规范化**：遵循第三范式（3NF）
- **性能优化**：合理冗余、索引优化
- **扩展性**：预留扩展字段
- **安全性**：敏感数据加密

### 1.2 Schema划分

```
public          # 公共表
auth            # 认证相关
course          # 课程相关
assessment      # 评估相关
content         # 内容相关
notification    # 通知相关
audit           # 审计日志
```

## 三、核心表设计

### 3.1 用户与权限（auth schema）

#### users表

```sql
CREATE TABLE auth.users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    email VARCHAR(255) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    first_name VARCHAR(100),
    last_name VARCHAR(100),
    avatar_url TEXT,
    status VARCHAR(20) DEFAULT 'active' CHECK (status IN ('active', 'inactive', 'suspended')),
    email_verified BOOLEAN DEFAULT false,
    last_login_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    deleted_at TIMESTAMP
);

CREATE INDEX idx_users_email ON auth.users(email);
CREATE INDEX idx_users_status ON auth.users(status);
```

### 3.2 课程管理（course schema）

#### courses表

```sql
CREATE TABLE course.courses (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    title VARCHAR(255) NOT NULL,
    description TEXT,
    teacher_id UUID NOT NULL REFERENCES auth.users(id),
    template_id UUID REFERENCES course.course_templates(id),
    cover_image_url TEXT,
    status VARCHAR(20) DEFAULT 'draft' CHECK (status IN ('draft', 'published', 'archived')),
    published_at TIMESTAMP,
    start_date DATE,
    end_date DATE,
    settings JSONB DEFAULT '{}', -- 课程设置（发布规则等）
    metadata JSONB DEFAULT '{}', -- 扩展元数据
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    deleted_at TIMESTAMP
);

CREATE INDEX idx_courses_teacher ON course.courses(teacher_id);
CREATE INDEX idx_courses_status ON course.courses(status);
CREATE INDEX idx_courses_dates ON course.courses(start_date, end_date);
```

#### course_templates表

```sql
CREATE TABLE course.course_templates (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(255) NOT NULL,
    description TEXT,
    category VARCHAR(100),
    structure JSONB NOT NULL, -- 课程结构模板
    is_public BOOLEAN DEFAULT false,
    created_by UUID REFERENCES auth.users(id),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### course_content表

```sql
CREATE TABLE course.course_content (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    course_id UUID NOT NULL REFERENCES course.courses(id) ON DELETE CASCADE,
    parent_id UUID REFERENCES course.course_content(id) ON DELETE CASCADE,
    type VARCHAR(50) NOT NULL CHECK (type IN ('module', 'lesson', 'quiz', 'assignment', 'resource')),
    title VARCHAR(255) NOT NULL,
    description TEXT,
    content JSONB, -- 内容数据（根据type不同结构不同）
    order_index INTEGER NOT NULL,
    duration_minutes INTEGER, -- 预计时长
    settings JSONB DEFAULT '{}', -- 发布设置、可见性等
    metadata JSONB DEFAULT '{}',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    deleted_at TIMESTAMP
);

CREATE INDEX idx_content_course ON course.course_content(course_id);
CREATE INDEX idx_content_parent ON course.course_content(parent_id);
CREATE INDEX idx_content_order ON course.course_content(course_id, order_index);
```

#### enrollments表

```sql
CREATE TABLE course.enrollments (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    course_id UUID NOT NULL REFERENCES course.courses(id) ON DELETE CASCADE,

    enrolled_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    enrolled_by UUID REFERENCES auth.users(id),
    status VARCHAR(20) DEFAULT 'active' CHECK (status IN ('active', 'completed', 'dropped', 'suspended')),
    progress DECIMAL(5,2) DEFAULT 0, -- 完成进度百分比
    last_accessed_at TIMESTAMP,
    completed_at TIMESTAMP,
    UNIQUE(course_id, student_id)

CREATE INDEX idx_enrollments_course ON course.enrollments(course_id);
CREATE INDEX idx_enrollments_student ON course.enrollments(student_id);
CREATE INDEX idx_enrollments_status ON course.enrollments(status);
```

#### standards表（教育标准库）

```sql
CREATE TABLE course.standards (

    parent_id UUID REFERENCES course.standards(id),
```

#### content_standards表（内容-标准关联）

```sql

    standard_id UUID NOT NULL REFERENCES course.standards(id) ON DELETE CASCADE,
```

### 3.3 评估系统（assessment schema）

#### assignments表

```sql
CREATE TABLE assessment.assignments (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    course_id UUID NOT NULL REFERENCES course.courses(id) ON DELETE CASCADE,
    content_id UUID REFERENCES course.course_content(id) ON DELETE CASCADE,
    title VARCHAR(255) NOT NULL,
    description TEXT,
    instructions TEXT,
    assignment_type VARCHAR(50) DEFAULT 'homework' CHECK (assignment_type IN ('homework', 'project', 'quiz', 'exam')),
    max_score DECIMAL(5,2) NOT NULL,
    passing_score DECIMAL(5,2),
    due_date TIMESTAMP,
    late_submission_allowed BOOLEAN DEFAULT false,
    late_penalty DECIMAL(5,2), -- 迟交扣分百分比
    max_attempts INTEGER DEFAULT 1,
    submission_types TEXT[] DEFAULT ARRAY['file', 'text'], -- 允许的提交类型
    file_types TEXT[], -- 允许的文件类型
    max_file_size INTEGER DEFAULT 10485760, -- 10MB
    rubric_id UUID REFERENCES assessment.rubrics(id),
    settings JSONB DEFAULT '{}',
    status VARCHAR(20) DEFAULT 'draft' CHECK (status IN ('draft', 'published', 'closed')),
    published_at TIMESTAMP,
    created_by UUID NOT NULL REFERENCES auth.users(id),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    deleted_at TIMESTAMP
);

CREATE INDEX idx_assignments_course ON assessment.assignments(course_id);
CREATE INDEX idx_assignments_due ON assessment.assignments(due_date);
CREATE INDEX idx_assignments_status ON assessment.assignments(status);
```

#### submissions表

```sql
CREATE TABLE assessment.submissions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    assignment_id UUID NOT NULL REFERENCES assessment.assignments(id) ON DELETE CASCADE,
    student_id UUID NOT NULL REFERENCES auth.users(id),
    attempt_number INTEGER DEFAULT 1,
    content TEXT, -- 文本内容
    file_urls TEXT[], -- 文件URL列表
    submitted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    status VARCHAR(20) DEFAULT 'submitted' CHECK (status IN ('draft', 'submitted', 'late', 'grading', 'graded', 'returned')),
    score DECIMAL(5,2),
    feedback TEXT,
    feedback_file_url TEXT, -- 反馈文件（如视频反馈）
    graded_by UUID REFERENCES auth.users(id),
    graded_at TIMESTAMP,
    rubric_scores JSONB, -- 评分标准各项得分
    time_spent INTEGER, -- 花费时间（秒）
    metadata JSONB DEFAULT '{}',
    UNIQUE(assignment_id, student_id, attempt_number)
);

CREATE INDEX idx_submissions_assignment ON assessment.submissions(assignment_id);
CREATE INDEX idx_submissions_student ON assessment.submissions(student_id);
CREATE INDEX idx_submissions_status ON assessment.submissions(status);
```

## 四、索引策略

### 4.1 主键索引

- 所有表使用UUID作为主键
- 自动创建B-tree索引

### 4.2 外键索引

- 所有外键字段创建索引

### 4.3 查询优化索引

```sql
-- 用户查询
CREATE INDEX idx_users_email_lower ON auth.users(LOWER(email));

-- 课程搜索
CREATE INDEX idx_courses_title_search ON course.courses USING gin(to_tsvector('english', title || ' ' || COALESCE(description, '')));

-- 内容搜索
CREATE INDEX idx_content_search ON course.course_content USING gin(to_tsvector('english', title || ' ' || COALESCE(description, '')));

-- 作业查询
CREATE INDEX idx_assignments_course_due ON assessment.assignments(course_id, due_date);

-- 提交查询
CREATE INDEX idx_submissions_assignment_status ON assessment.submissions(assignment_id, status);
```
